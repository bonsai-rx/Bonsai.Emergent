## About

`Bonsai.Emergent` provides operators for acquiring images from Emergent Vision cameras. [eSDK](https://emergentvisiontec.com/software/esdk/) version 2.62.02 is required to build and use this library.

## How to Use

To use `Bonsai.Emergent` for visual reactive programming, please install this package using the [Bonsai package manager](https://bonsai-rx.org/docs/articles/packages.html).

## Documentation

For additional documentation and examples, refer to the [official Bonsai documentation](https://bonsai-rx.org/emergent/).
